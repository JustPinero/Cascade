# Universal Project Kickoff Template — Web App v3.3

This is the default kickoff template for web application projects.

## Sections

1. **Project Identity** — App name, tagline, problem statement, experience description
2. **Stack Definition** — Frontend, backend, language, database, auth, hosting, testing
3. **Core Data Model** — Primary entity, key fields, relationships, secondary entities
4. **Phase Breakdown** — Phased development plan with exit criteria
5. **Environment Variables** — Required and optional secrets
6. **Known Constraints** — Technical and business constraints
7. **Design Direction** — Aesthetic, color palette, typography, UI details
8. **Directory Structure** — Full project file tree
9. **Testing Protocol** — Test-driven development approach
10. **Debt Management** — Technical debt tracking

## Usage

This template is used by the Cascade project creation wizard. Claude will interview the developer using the structure above and produce a filled-in kickoff prompt tailored to their specific project.

## Customization

Edit this template or create new ones via the Template Management page in Cascade.
