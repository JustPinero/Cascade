# Phase 21 — Conversation Mode + Push-to-Talk

## Why
Phase 20 gave Delamain a voice. The chat already had basic STT (the
Mic button). What was missing: chaining the pieces into actual
back-and-forth dialogue. Phase 21 closes the loop.

## Decisions locked
- One **Conversation** toggle on the chat screen. Always off on page
  load. Off on nav-away / unmount (chat owns the state, not settings).
- When toggling off mid-utterance: **let TTS finish, disarm the mic.**
- **Silence threshold** configurable in settings (default 1500ms).
- **Mic input mode** in settings: `toggle` (existing) or
  `push-to-talk` (hold to record, release to auto-submit). Same Mic
  button, mode-dependent.
- Conversation Mode overrides mic-mode setting when active.
- Listening indicator: red dot + chat-bubble icon + "Listening...".
- Esc key: cancels TTS, stops mic, exits Conversation Mode.
- Typing in input or clicking Send while listening: interrupts mic,
  submits manually.
- "Use talking portrait" checkbox in Overseer settings; hides the
  talking-portrait URL field when off.

## Sub-requests

### 21.1 — Settings + silence-detector helper
- Extend `OverseerSettings` with `usesTalkingFace`, `silenceThresholdMs`,
  `micMode`.
- Settings UI: talking-face checkbox, silence slider, mic-mode radio.
- New `lib/silence-detector.ts` — pure timer state-machine. Testable
  with `vi.useFakeTimers`.

### 21.2 — Conversation Mode core
- New chat-screen button "Conversation". Always off on mount.
- When ON: SpeechRecognition runs continuous + interim. Interim
  text appears in input as you speak. Silence-detector auto-submits
  after threshold.
- After Del's TTS `onComplete`, mic auto-reopens (unless toggled
  off in the interim).
- While Del is speaking, mic is force-disabled (no echo loop).
- Cleanup hook: on unmount or toggle-off, stop mic. (TTS in flight
  is allowed to finish.)

### 21.3 — Push-to-Talk
- Mic button: when `micMode === "push-to-talk"`, uses onMouseDown/Up
  + onTouchStart/End. Hold to record; release auto-submits.
- When `micMode === "toggle"`: existing click toggles unchanged.

### 21.4 — Polish
- Listening indicator in chat header: red dot + chat-bubble icon
  + "Listening..." label. Shown when mic is actively listening.
- Esc key handler bound while chat is mounted: cancels TTS, stops
  mic, exits Conversation Mode.
- Typing interrupt: input change detected as user-driven (not from
  interim transcript) stops the mic and submits manually.

## Out of scope
- Whisper / cloud STT
- Wake word ("Hey Delamain")
- Custom keyboard shortcuts (only Esc this phase)
- Multi-language voice picker
