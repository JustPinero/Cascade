# Request 7.1 — Delamain Sound Effects

## Objective
Give Delamain a voice through sound effects. Short, distinct audio cues for key events so the developer knows what's happening without looking at the screen.

## Acceptance Criteria
- [ ] 3-4 sound effects sourced from open-source/royalty-free audio
- [ ] Sounds play via Web Audio API in the browser (no server-side audio)
- [ ] Sound triggers: (1) Delamain starts responding, (2) Delamain finishes responding, (3) escalation/blocker notification
- [ ] Sounds respect a mute toggle in Settings (stored in localStorage)
- [ ] Sounds are short (< 1 second) and non-intrusive
- [ ] Sound files stored in /public/sounds/
- [ ] No sounds play if browser tab is not focused (avoid surprise audio)

## Files to Touch
- `public/sounds/` — new audio files
- `lib/sounds.ts` — new, audio playback utility
- `app/components/overseer-chat.tsx` — trigger sounds on stream start/end
- `app/components/activity-feed.tsx` — trigger sound on escalation notification
- `app/settings/page.tsx` — add mute toggle

## Tests to Write
- `lib/sounds.test.ts` — test mute preference, sound file existence validation
- Verify sounds don't play when muted
- Verify sounds don't play when tab is not focused

## Dependencies
- None
