# Request 7.2 — Delamain RPG Portrait & Talking Animation

## Objective
Display Delamain's portrait at the top of the /delamain chat page like an old-school RPG dialogue box. Animate the portrait while Delamain is responding to simulate "talking."

## Acceptance Criteria
- [ ] Portrait displayed prominently at top of /delamain page (not just the small header icon)
- [ ] Portrait styled like an RPG dialogue frame — bordered, name plate below, cyberpunk aesthetic
- [ ] Talking animation plays while `streaming` is true (Delamain is responding)
- [ ] Animation stops cleanly when response completes
- [ ] Animation approach: CSS-based (pulse/glow OR mouth open/close if two image assets provided)
- [ ] Portrait is static/idle when Delamain is not responding
- [ ] Responsive — portrait scales appropriately on smaller screens
- [ ] Name plate shows "DELAMAIN" in the project's mono/cyberpunk font

## Files to Touch
- `app/components/overseer-chat.tsx` — add portrait section when `fullPage` is true
- `app/globals.css` — add talking animation keyframes if needed
- `public/delamain.jpg` — existing portrait (may need a second "talking" variant)

## Tests to Write
- Component renders portrait when fullPage=true
- Portrait has idle class when not streaming
- Portrait has talking class when streaming
- Portrait does not render when fullPage=false (dashboard embed)

## Dependencies
- Depends on 7.1 (sounds complement the animation)

## Design Notes
Think: Final Fantasy dialogue box, Chrono Trigger character portraits, or Fallout terminal with a face. The portrait is a visual anchor — when Del is "talking," the user should feel like they're in a conversation with an entity, not a text box.

If only one image asset is available, use CSS effects for the talking state:
- Pulsing cyan glow ring around portrait
- Subtle scale oscillation (1.0 → 1.02 → 1.0)
- Scanline effect intensifies during speech
