# Request 2.2 — Health Derivation Engine

## Objective
Build the engine that reads audits/debt.md, git status, and test results from each project to compute health scores.

## Acceptance Criteria
- [ ] Reads debt.md and counts open items
- [ ] Checks git status for uncommitted changes
- [ ] Parses recent audit results
- [ ] Computes overall health (healthy/warning/blocked/idle)
- [ ] Handles missing files gracefully
- [ ] Returns structured HealthResult
