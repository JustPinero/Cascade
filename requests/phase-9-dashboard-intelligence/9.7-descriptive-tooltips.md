# Request 9.7 — Descriptive Tooltips

## Objective
Add hover tooltips to sidebar nav items and dashboard action buttons
so new users understand what each feature does without documentation.

## Acceptance Criteria → Test Mapping

| Criterion | Test |
|-----------|------|
| Sidebar nav items show tooltip on hover | NavLink renders title attribute |
| Dashboard buttons show tooltip on hover | ScanButton, HarvestAll, ResumeAll have tooltips |
| Brief Me button has tooltip | MorningBriefing button has title |
| Tooltips are concise (under 60 chars) | all tooltip strings under 60 chars |
| Tooltips don't interfere with click targets | hover delay prevents accidental show |

## RED Phase
- NavLink component accepts and renders tooltip prop
- Dashboard buttons have title attributes
